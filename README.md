# 项目简介

基于深度学习的施工现场安全检测系统，融合了web界面与YOLOv8至YOLOv12系列模型，旨在提升施工现场安全管理效率。该系统主要功能模块包括实时视频分析、隐患自动识别与警报、安全数据统计等，技术特点在于运用了最新的深度学习算法，对施工人员行为、安全帽佩戴、危险区域入侵等安全问题进行有效检测。此系统不仅提高了检测的准确率，也减少了人工监管的强度，有助于构建更为安全的施工环境。


## 核心功能
- 实时视频分析：对施工现场视频流进行实时监控与分析
- 隐患自动识别与警报：自动识别施工人员行为异常、安全帽佩戴情况及危险区域入侵等安全隐患，并触发警报
- 安全数据统计：对检测结果进行统计与分析，生成安全报告
- 支持多种YOLO模型：兼容YOLOv8至YOLOv12系列深度学习模型

## 适用场景
- 建筑施工现场安全管理
- 施工人员行为规范监督
- 安全防护装备佩戴检查
- 危险区域入侵监控
- 施工安全数据分析与管理

## 技术特点
- 基于深度学习算法，提升检测准确率
- Web界面设计，便于操作与管理
- 自动化检测减少人工监管强度
- 构建更安全的施工环境
